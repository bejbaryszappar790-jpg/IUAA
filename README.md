IUAA: InVision U Admission Assistant 🚀
Интеллектуальная система автоматизированного пре-скрининга и экспертной оценки абитуриентов на базе DeepSeek-R1.

Разработано в рамках хакатона Decentrathon 5.0 (Трек: AI inDrive / Invision U).

📌 Обзор проекта
IUAA — это инновационное решение для приемных комиссий, позволяющее объективно оценивать потенциал кандидатов в условиях массового использования генеративного ИИ. Система анализирует не форму, а содержание, выявляя реальные достижения и Soft Skills.

Ключевые возможности:

🛡 Skeptical AI Evaluator: Уникальный алгоритм на базе промпт-инжиниринга, настроенный на поиск конкретных доказательств (Evidence-based scoring) и детекцию шаблонных ответов LLM.

📄 Интеллектуальный парсинг документов: Полная поддержка форматов PDF и DOCX. Система автоматически извлекает текст, очищает его от шума и передает в ИИ-ядро.

⚖️ Скоринг по критериям InVision U: Детерминированная оценка по 10-балльной шкале для метрик: Лидерство, Ориентация на рост и Подлинность.

⚡ Production-ready Backend: Асинхронная архитектура на FastAPI, обеспечивающая высокую пропускную способность и отказоустойчивость.

🏗 Архитектура системы (Clean Architecture)
Проект реализован по модульному принципу для легкой масштабируемости:

src/api/: Транспортный слой. Обработка HTTP-запросов, валидация через Pydantic и управление эндпоинтами.

src/ai_modules/: Ядро системы. Логика оценки (evaluator.py), экстракция текста (file_reader.py) и база системных инструкций (prompts.py).

data/: Хранилище синтетических данных и логов для калибровки моделей.

models/: Конфигурации для локального запуска LLM (Ollama/DeepSeek).

🛠 Технологический стек
Backend: Python 3.12+, FastAPI, Uvicorn, Pydantic.

AI Core: DeepSeek-R1 (7B) via Ollama API.

Document Processing: pdfplumber, python-docx.

Infrastructure: Git, Virtualenv.

🚀 Быстрый старт

1. Подготовка окружения

# Создание и активация виртуального окружения
python3 -m venv .venv
source .venv/bin/activate

# Установка всех необходимых библиотек
pip install -r requirements.txt

2. Запуск сервера

# Запуск из корня проекта
uvicorn src.api.main:app --reload

Интерактивная документация API (Swagger) доступна по адресу: http://127.0.0.1:8000/docs

Почему наше решение эффективно?

В условиях, когда каждый абитуриент может использовать ChatGPT для написания эссе, стандартные методы отбора перестают работать. IUAA меняет парадигму: мы используем мощность DeepSeek-R1, чтобы находить "зерна" реального опыта в "плевелах" сгенерированного текста, обеспечивая университету выбор действительно талантливых студентов.
